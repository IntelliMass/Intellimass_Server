# Intellimass Server

### Run Server Localy

python iapp.py

#### Localhost Routes

    1. http://localhost:5000/query          [POST]
    2. http://localhost:5000/articles       [GET]
    3. http://localhost:5000/metadata       [GET]
    4. http://localhost:5000/networks       [GET]
    5. http://localhost:5000/categories     [GET]
